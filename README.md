# Locadora
Teste Técnico - GRUPO SOITIC

Este projeto foi desenvolvido em `<c++ cli>` e usando `<SQLite>`. O usuário e senha de acesso são ambos `<Admin>`.

### Contem os seguintes forms:

* AddDirectorForm
* AddMovieForm
* AlterDirectorForm
* AlterMovieForm
* DeleteDirectorForm
* DeleteMovieForm
* MainForm
* SearchDirectorForm
* SearchMovieForm

### As seguintes Classes:

* User
* Movie
* Director
* Controller

### Dependências:

* `<System.Data.SQLite;>`
